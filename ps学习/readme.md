这就像你用一碳把一张白纸涂成了黑色。白色就是背景色，不用橡皮擦是看不见的；黑色是前景色

## 渐变

1、选中矩形工具头部就会出现填充菜单

2、选中要编辑的图层，然后设置填充即可

3、设置好了，选中图层，鼠标左键按住划线即可，划线的方向就是渐变的方向
